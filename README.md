# burningplane911
Сучасна компанія, яка допомагає легко забронювати подорож у будь-який куточок світу. Ми відкриваємо двері до нових країн, культур і незабутніх пригод лише в кілька кліків. Подорожуйте без зайвих турбот — ми подбаємо про кожну деталь вашої мрії.

Реалізовано:
- сторінку входу користувача
- поля для введення логіну та пароля
- кнопку "Увійти" для авторизації
- кнопку "Реєстрація" для створення нового акаунту
Після успішного входу користувач отримує доступ до функцій системи.

На головній сторінці розташовано: 
~ логотип сервісу
~ навігаційне меню:
    - Головна
    - Гарячі тури
    - Пошук тура
    - Про нас
    - Контакти
    - Профіль

Стек технологій:
Для створення інтерфейсу використано:
- HTML - структура сторінки
- CSS - оформлення дизайну
- Figma - створення прототипу інтерфейсу
